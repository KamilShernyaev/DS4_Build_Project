# Unity Dark Souls Build

1. Download archive DS4_Build_Project.rar
2. Run Dark Souls 4.exe
3. Try to PLay

## Control
- WASD
- Jump: Space
- Lock On/Off: N
- Simple Attack: E
- Hard Attack: R
- Roll: Left Shift
- Sprint: Hold Shift
- Inventory: Escape
- Pick Up: F
- Quick Slots: Arrow
- Flask: X
- Shield: Q
- Weapon Two Hand: Y
